# Simple-Shutdown-Timer-with-GUI

Simple Windows shutdown timer with GUI, presets, tray icon, and live countdown. Set custom hours and minutes or use quick presets. Always-on-top option included. Written with Python.

---------------------------------------------------------------------

Simple timer to shut down Windows. Uses the Windows shutdown.exe. Written to run through my cheap AliExpress Stream Deck to shut down my PC at a specific time.
